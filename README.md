# Go-Lambda-To-Kakao
CloudWatch Alarm을 Lambda로 이용하여 카카오톡으로 알람 전송 (Go 언어로 작성)

AWS EC2(Amazon Linux)에서 람다 함수를 압축한 후에 S3에 전송 후 람다에 올립니다.
